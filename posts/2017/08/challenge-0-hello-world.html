Hey guys :)

Here is the first challenge in my Android Course: Make a Hello World! App using Android Studio. You can find the project's repo here or download it from here.

But I wanted to build something a little more fancy than a simple Hello World! project, which is created by default when you start a new project in Android Studio, so I search in the App Developer Resources page from Google, and I found an official tutorial for building your first app.

In the app you enter a name and the app prints in the screen:

Hello <your_name> to the World!

Building this app, I learned something new: You can add HTML content to a TextView :O! You can do it using the fromHtml method which is in the HTML class.

Here's the snippet:

@SuppressWarnings("deprecation")
public static Spanned fromHtml(String source) {
    if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.N) {
        return Html.fromHtml(source, Html.FROM_HTML_MODE_LEGACY);
    } else {
        return Html.fromHtml(source);
    }
}

And here's how to use it:

Spanned message = fromHtml("Hello <b>"+extra_message.toUpperCase()+"</b> to the World!");
