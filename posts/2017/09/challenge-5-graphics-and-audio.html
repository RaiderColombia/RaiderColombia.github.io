Jhon Ramirez | Mon 18 September 2017

Hey buddies!

In this tutorial the challenge is to build the mechanism to paint the graphics and play sounds in the TicTacToe game I'm doing for my Android Course at College.

You can find the project's repo here or download a zip file with the source code from here.

To solve the challenge I'm following this tutorial in which the author (Frank McCown) explains how to paint graphics using a BoardView class which extends from the View class and overrides the onDraw() method to control the way the game is rendered. Also, in the tutorial you can find how to play sounds using the MediaPlayer class. More info about MediaPlayer here.

The cool thing I've learnt with this challenge was How to play sounds in an android app?

To play sounds you have to use the MediaPlayer class. In my case I will play a sound every time a player makes a move.

In first place, you have to define a MediaPlayer object.

private MediaPlayer mHumanMediaPlayer;

Later you have to load the sound effects into the media player when the activity goes into the Resume state, and release it when the activity goes into the Pause state.

@Override
protected void onResume(){
    super.onResume();
    mHumanMediaPlayer = MediaPlayer.create(getApplicationContext(), R.raw.player_move);
}

@Override
protected void onPause(){
    super.onPause();
    mHumanMediaPlayer.release();
}

And finally you have to start playing the sound effect.

mHumanMediaPlayer.start();

That's it for now!
